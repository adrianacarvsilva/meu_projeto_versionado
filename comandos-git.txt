
Inicializar Novo Repositório
	$git init

Status do Repositório
	$git status

Adicionar os arquivos que foram modificados para fazer o commit posteriormente por nome
	$git add 'index.html'

Remover os arquivos - servidor
	$git rm index.html

Remover os arquivos - force (local)
	$git rm -f index.html

Commit nos arquivos
 	$git add . (add todos) git add 'nomedoarquivo.extensao'
	$git commit -m 'descrição do commit'

Após alterar um arquivo e desejar retornar para a versão anterior
	$git checkout -- 'index.html'  

Usuários e senhas para autenticar automaticamente
	$git config --global credential.helper 'store'
	$git git-credentials

Conectando repositórios locais com repositórios remotos
	$git remote 
	$git remote -v (verifica se já existe diretório cadastrado com mesmo nome)
	$git remote add origin git@github.com:adrianacarvsilva/meu_projeto_versionado.git

Enviando alterações para o repositório remoto
	$git push --set-upstream origin master
	$git push -f origin master

Alterando o valor do repositório remoto configurado
	$git remote set-url origin 'novaurl'

Clonando um repositório
	$git clone url

Atualizando o ambiente local com os commits do servidor
	$git pull
