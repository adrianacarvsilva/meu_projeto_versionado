
Inicializar Novo Repositório
	$git init

Status do Repositório
	$git status

Adicionar os arquivos que foram modificados para fazer o commit posteriormente por nome
	$git add 'index.html'

Remover os arquivos - servidor
	$git rm index.html

Remover os arquivos - force (local)
	$git rm -f index.html

Commit nos arquivos
 	$git add . (add todos) git add 'nomedoarquivo.extensao'
	$git commit -m 'descrição do commit'

Após alterar um arquivo e desejar retornar para a versão anterior
	$git checkout -- 'index.html'  

Usuários e senhas para autenticar automaticamente
	$git config --global credential.helper 'store'
	$git git-credentials
